# java基本数据类型
java变量在创建时通过申请内存储存值，内存根据变量的类型分配储存空间，并且该空间只能储存相同类型的值。
- java分为两大数据类型：内置数据类型/引用类型
  - 内置数据类型：内置数据类型包括八种基本数据类型
    - byte
      - 8位，有符号的以二进制补码表示的整数
      - 取值范围为`-128~127 （-2^7 ~ 2^7-1）`
      - 默认值为`0`
      - byte在大型的数组中替代整数可以节约空间，byte占用的控制只有int的四分之一
    - short
      - 16位，有符号的二进制补码表示的整数
      - 取值范围为`-32768~32767 (-2^15 ~ 2^15-1)`
      - 默认为`0`
    - int
      - 32位，有符号的二进制补码表示的整数
      - 取值范围为`-2147483648~2147483647（-2^31 ~ 2^31-1）`
      - 默认值为`0`
    - long
      - 64位，有符号的二进制补码表示的整数
      - 取值范围为：`-2^63~2^63-1`
      - 默认值是`0L`
    - float
      - 单精度，32位，符合IEEE 754标准的浮点数
      - 默认值为`0.0f`
      - 浮点数不能表示精确的数值，会有误差
    - double
      - 双精度，64位，符合IEEE 754标准的浮点数
      - 浮点数的默认类型位double
      - 默认值为`0.0d`
    - boolean
      - 表示一位的信息，取值为true/false
      - 默认为false
    - char
      - 是一个单一的16位的Unicode字符
      - 取值范围位`\u0000~\uffff`即为`0~65535`
    - PS:实际上java中的void也是一种基本数据类型，也有对应的包装类java.lang.void，但我们无法直接操作
  - 引用类型：
    - java中的引用类型类似与C++中的指针，引用类型指向一个变量，指向的变量成为引用变量，引用变量在声明时被确定一个特定的类型，声明之后，应用变量的类型不可改变
    - 引用类型的默认值为null
    - 对象、数组、枚举等都为引用类型
    
  - 基本数据类型和引用数据类型区别：
    - 基本数据类型在被创建的时候在栈上分配一块内存，将数值存储在栈上。
    - 引用数据类型创建时先在栈上给他的引用分配一块内存，而将引用的对象存储在堆内存中，然后栈上的引用指向堆中的地址
    - UserVO user = new UserVO("Xiaoming");语句在内存中创建的过程：
      - 在栈内存为user分配一块空间
      - 在堆中为UserVO类的对象分配一块空间，并为UserVO类中的属性先赋予默认值：例：userName = “”
      - 根据构造方法，将属性赋值userName=“Xiaoming”
      - 将User对象在堆中的地址赋值给栈中的user，通过引用user可以找到在堆中对象的具体信息

- Java自动类型转换
  - 各种数据类型可以混合运算，但需要转换成统一类型，然后进行运算。
    
        转换由低级到高级可以自动转换：
        byte,short,char―> int ―> long―> float ―> double 
        
  - java中转换必须满足：
    - boolean类型不能转换
    - 容量大的类型转换成容量小的类型必须要强制类型转换
    - 对象之间的转换数据类型必须兼容
    - 转换过程中会损失精度
    - 浮点数转换为整数是通过舍弃小数位，而不是四舍五入
