# Java简介
Java是由sun公司1995年推出的高级编程语言。2010年被Oracle收购。
- 主要特性：
  - 面向对象：封装，继承，多态
  - 健壮性：强类型，异常处理，垃圾收集
  - 跨平台：基于jvm，支持多个平台
  - 解释型语言
  - 高性能
  - 多线程
- 术语：
  - JDK（Java development kit）：Java开发工具
  - JRE（Java runtime environment）：Java运行时环境
  - JVM（Java virtual machine）：Java虚拟机，提供java程序运行平台，将字节码转换成机器码。Java程序跨平台的基础
