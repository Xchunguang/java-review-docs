# 进制之间的转换
- 进制是什么：
  进制就是进位计数制，是一种规定的进位方法。对于任何一种X进制，都是逢x进一位。
- 进制的用途：
  我们在使用高级语言编程时，通常使用的是10进制进行编程。不过在计算机中的表示最终是以二进制的形式存在，所以有时使用二进制更直观的解决问题。但是二进制表示太长，表达一个int值需要32位，如：
  
      0000 0000 1111 1111 0000 1111 1111 0000
      
  由此，编程语言提供使用8进制或16进制解决这个问题。所以一般来说，2进制是计算机内部使用，10进制是人们使用，16进制则是2进制方便阅读的记录形式。
- 常用的进制：
  - 二进制（Binary）：0b0001
  - 十进制（Decimal）：01
  - 十六进制（Hexadecimal） ：0x01
- 进制之间的转换：
  - 各种进制之间都可以进行转换，但都可使用以下方式进行计算：
    - 十进制转X进制
    - X进制转10进制
  - 十进制转X进制：
    - 整数部分：
      十进制数除以X，余数倒序即为X进制
      
          例：十进制98转换位2进制
          98 ÷ 2 = 49 余 0
          49 ÷ 2 = 24 余 1
          24 ÷ 2 = 12 余 0
          12 ÷ 2 = 6  余 0 
          6  ÷ 2 = 3  余 0
          3  ÷ 2 = 1  余 1
          1  ÷ 2 = 0  余 1
          所以98的二进制位1100010
          
    - 小数部分：
      小数与X相乘，结果取出整数位，小数部分继续与X相乘，直到结果为0。取出的整数位正序即为X进制。不是所有的十进制小数位都可以转换成X进制小数位，实际使用根据精度计算。

          例：十进制0.85转换成2进制
          0.85 × 2 = 1.7 取整数位 1
          0.7  × 2 = 1.4 取整数位 1
          0.4  × 2 = 0.8 取整数位 0 
          0.8  × 2 = 1.6 取整数位 1 
          0.6  × 2 = 1.2 取整数位 1
          0.2  × 2 = 0.4 取整数位 0
          ...
          所以，0.85转换位2进制为无限循环的。当前精度下结果取正序：0.85 = 0.110110B
          
  - X进制转10进制：
    权数求和：权数是指当前位数从右至左的从0开始的次序号，小数位从-1开始往右。
    
      例：X进制12345转换为10进制：
      5 × X<sup>0</sup> + 4 × X<sup>1</sup> + 3 × X<sup>2</sup> + 2 × X<sup>3</sup> + 1 × X<sup>4</sup>
      
  - 负数十进制转换成二进制：
    - 整数部分：
      取整数绝对值 -> 使用整数的绝对值得到的二进制码为原码 -> 原码转换为反码 -> 反码加1即为整数部分的二进制数，称为补码
 
          例：-5的2进制
          -5 转换为 5
          5的二进制为 101 ，则原码为101，八位二进制位00000101
          原码取反，反码为 11111010
          反码加1，即为结果（补码）：11111011
          
          例：二进制负数11111011转换为十进制数：
          补码减1，得到反码：11111010
          反码取反，得到原码：00000101
          原码转换为10进制，得到5
          取反，结果为 -5
           
          
  - 负数十进制如何转换为八进制和十六进制
    将十进制转换为二进制补码，然后三位一断得到八进制，四位一断得到十六进制
